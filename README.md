#Difference between getElementById and getElementsByClassName:
getElementById:
1.Select by Id name
2.Return single element 
3.unique
4.Access specific element

getElementByClassName:
1.Select by Class name
2.Return HTML collection
3.Not unique
4.Access multiple elements
#Difference between querySelector and querySelectorAll:
querySelector:
1.Return first matching element
2.Do not support loop
3.Behavior likes Id

querySelectorAll:
1.Return all matching elements as a NodeList
2.Support loop
3.Behaviour likes Class
